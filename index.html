Surgimento da Internet

voltar para o inicio

Com as grandes guerras reconrrentes da primeira e segunda guerra principalmente forçou as nações a correrem em um corrida armamentista
e tecnológica muito forte durante e após a 2° guerra mundial. Com isso as nações desenvolveram as primeiras máquinas semi-mecânicas como exemplo da Mark1.
Após isso houve a necessidade de receber e enviar dados de um determidado lugar para outro, foi onde em 1960 se viu criada a ARPANET com a finalidade de enviar informações entre centros de pesquisas dos EUA.

A primeira mensagem envianda foi entre dois computadores de UCLA e da SRI onde foi envida a mensagem "LO". Dando início a história deste marco revolucionário para a humanidade.

Como Funciona o HTML

O HTML ou Hyper Text Markup Language não é uma linguagem de programação mas uma ferramenta incrível de criação e estruturação de sites ou aplicações utilizando hipertextos

O que são tags

voltar para o inicio

As tags em HTML são como guias ou palavras chaves para desiginar comandos para a formatação do texto que será exibido na página web ou aplicação. Como guias cada comando tem a sua particularidade e função veja exemplos:

  • H1 - header (cabeçario) = TÍTULO - nível muito maior
  • H2 - header (cabeçario) = SUBTÍTULO - nível maior
  • H3 - header (cabeçario) = nível maior um pouco
  • H4 - header (cabeçario) = nível médio
  • H5 - header (cabeçario) = nível menor um pouco
  • H6 - header (cabeçario) = nível menor
  • < p > - parágrafo = texto normal
  • < br > - quebra de linha"
  • < hr > - linha horizontal
  • < blockquote > - citação"
  • < u > - underline"
  • < mark > - marca texto
